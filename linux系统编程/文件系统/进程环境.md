### 1. 从main函数返回
```c
#include <stdio.h>
int main(){
    printf("helloworld");
    return 0;  
}
```
在 C 语言中，`return 0;` 是用来表示程序正常退出的一种约定，它的主要目的是向调用程序（通常是操作系统或父进程）返回一个退出状态码。
当程序运行结束后，操作系统（或父进程）会通过返回值来判断程序是否正常退出。
在标准约定中：
返回值 0 表示程序正常退出,非零值（通常是正整数）表示程序发生了某种错误，具体错误含义由程序员定义。
